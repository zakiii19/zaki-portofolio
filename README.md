# Qolbul Muzaki — Portfolio

> Fresh Graduate IT dengan keahlian Design Thinking, Prototyping, dan User Centered Design

Portfolio satu halaman yang dibangun dengan **HTML5 semantik**, **CSS3 modern**, dan **desain neobrutalism**. Responsive mobile-first untuk recruiter, tech lead, dan klien freelance.

## 🎨 Features

- **Semantic HTML5**: Struktur yang aksesibel dan mudah dipahami
- **Mobile-First**: Responsive dari 320px hingga desktop
- **Neobrutalism Design**: Border tegas, shadow block, kontras kuat
- **Zero Non-Semantic Divs**: Menggunakan elemen semantik (header, nav, main, section, article, footer)
- **Hamburger Menu**: Mobile navigation yang rapi dan fungsional
- **Accessibility Basics**: aria-label, focus-visible, dan semantic landmarks

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Mobile-first responsive design

## 📂 File Structure

```
index.html      # Semantic HTML structure
style.css       # Responsive CSS with neobrutalism
README.md       # Project documentation
plan/           # Planning & execution documents
├── 01-brainstorm.md
├── 02-details.md
├── 03-execution.md
└── 04-results.md
```

## 🎯 Sections

- **Hero**: Nama, tagline, dan CTA
- **About**: Background dan skills utama
- **Skills**: HTML5 Semantik, CSS Modern, Neobrutalism
- **Work**: Project showcase
- **Contact**: Email dan LinkedIn

## 🚀 How to Use

1. Clone atau download repository
2. Buka `index.html` di browser
3. Test responsivitas di mobile dan desktop

## 📱 Responsive Breakpoints

- **Mobile**: 320px - 599px (hamburger menu, stacked layout)
- **Tablet+**: 700px+ (horizontal nav, grid layout)

Proyek ini mendemonstrasikan pemahaman tentang:
- HTML5 semantik untuk aksesibilitas
- CSS modern dengan responsive design
- Desain neobrutalism yang bold namun tetap bersih
- Mobile-first approach
